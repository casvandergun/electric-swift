# ElectricSwift

Low-level Swift runtime and protocol types for ElectricSQL shape streams.

## Package Contents

- `ShapeStream` as the primary stream runtime
- Electric protocol models, message decoding, and shape state
- SSE and long-poll stream coordination
- Shape request construction and recovery helpers
- `MaterializedShape` for non-SwiftData consumers

## Notes

- This export bundle was prepared from `electric-swift-data`.
- `ShapeStream` is the public runtime name.
